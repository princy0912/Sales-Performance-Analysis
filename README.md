# Sales-Performance-Analysis
Sales Data Analysis Project using Python, Pandas, NumPy and Matplotlib
# 📊 Sales Performance Analysis

## 🧠 Project Overview
This project analyzes sales data from the Superstore dataset using Python, Pandas, NumPy, and Matplotlib.  
The goal is to generate business insights such as top products, regional performance, profit analysis, and visualizations that help stakeholders make data-driven decisions.

---

## 🛠 Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter / VS Code

---

## 📁 Dataset
The dataset used is **SampleSuperstore.csv** which includes:
- Sales
- Profit
- Region / City
- Product and Sub-Category
- Ship Mode, etc.

Total dataset size: **9994 rows × 13 columns**

---

## 🔍 Key Analysis Performed

### 📌 Data Cleaning
- Removed duplicate rows
- Checked for missing values
- Converted date fields

## 📌 Recommendations
Based on the analysis:
1. Focus more on high-profit sub-categories to increase revenue.
2. Reduce deep discounts on loss-making sub-categories.
3. Improve pricing strategy for products with negative profit margin.
4. Target marketing in high performing regions and cities.
5. Optimize supply chain for loss-making states.
